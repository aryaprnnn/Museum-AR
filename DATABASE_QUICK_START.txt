## ğŸ†˜ JIKA ADA ERROR

### **Error 1: "Can't connect to MySQL server"**
**Solusi:** Start MySQL di XAMPP Control Panel

### **Error 2: "Access denied for user 'root'@'localhost'"**
**Solusi:** Cek `.env` â†’ ubah `DB_PASSWORD` sesuai password MySQL Anda
(Default XAMPP: kosong = `DB_PASSWORD=`)

### **Error 3: "Unknown database 'museum_ar'"**
**Solusi:** Buat database dulu di phpMyAdmin sebelum jalankan `php artisan migrate`

### **Error 4: SQLSTATE error saat migrate**
**Solusi:** 
```bash
# Reset semua
php artisan migrate:fresh

# Atau di phpMyAdmin, hapus semua tabel dan jalankan migrate lagi
```
